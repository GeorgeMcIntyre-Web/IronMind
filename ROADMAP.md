# Roadmap

## Milestone M1: MVP (complete)
- Stable API surface for run lifecycle.
- Deterministic stub provider for offline testing.
- IR schema and validation with clear errors.
- CP-SAT solving with audit trail.

## Milestone M2: CI + Dev Quality (complete)
- GitHub Actions workflow for tests.
- Developer checklists for safe changes.

## Milestone M3: Expand (future)
- More IR constructs if demanded by users.
- Better telemetry on provider output quality.
- Optional performance reporting for solver runs.
